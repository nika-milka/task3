# Настройки запуска и базовая защита веб службы

## 📝 Описание проекта

Проект представляет собой веб-службу на базе ASP.NET Core с гибкой системой конфигурации и многоуровневой защитой. Приложение поддерживает чтение настроек из трёх источников (файл, переменные окружения, аргументы командной строки) с явным приоритетом, выполняет раннюю валидацию конфигурации и обеспечивает базовую безопасность через проверку доверенных источников, ограничение частоты запросов и защитные HTTP-заголовки.

## 🎯 Цель работы

Реализовать веб-службу с настраиваемой конфигурацией и базовой защитой:

- Чтение настроек из трёх источников с явным порядком приоритета
- Ранняя проверка корректности настроек с запретом запуска при ошибках
- Ограничение запросов только с доверенных источников
- Ограничение частоты запросов для защиты от DoS-атак
- Добавление защитных заголовков ответа
- Два режима работы (учебный и боевой) с переключением через настройку

## 🏗️ Компоненты системы

### Ядро приложения

| Компонент | Описание |
|-----------|----------|
| `Program.cs` | Точка входа, настройка конфигурации и middleware |
| `AppSettings` | Модель конфигурации с режимом, доверенными источниками и лимитами |
| `ConfigValidator` | Валидация настроек перед запуском приложения |

### Middleware (промежуточное ПО)

| Компонент | Описание | Функции |
|-----------|----------|---------|
| `SecurityHeadersMiddleware` | Безопасность ответов | Проверка Origin, защитные заголовки |
| `RateLimitingMiddleware` | Ограничение запросов | Разные лимиты для разных маршрутов |

### Модели данных

| Модель | Поля | Описание |
|--------|------|----------|
| `AppSettings` | Mode, TrustedOrigins, RateLimits, EnableDetailedErrors | Основная конфигурация |
| `RateLimitSettings` | GeneralRequestsPerMinute, CreateRequestsPerMinute | Лимиты запросов |

## ⚙️ Функциональность

### Основные возможности

| Функция | Статус | Описание |
|---------|--------|----------|
| ✅ Чтение из 3 источников | Реализовано | Файл → Env → CLI |
| ✅ Приоритет настроек | Реализовано | CLI переопределяет всё |
| ✅ Ранняя валидация | Реализовано | Проверка перед запуском |
| ✅ Запрет запуска при ошибках | Реализовано | Понятные сообщения об ошибках |
| ✅ Проверка доверенных источников | Реализовано | Только разрешённые Origin |
| ✅ Rate Limiting | Реализовано | 60/мин GET, 10/мин POST |
| ✅ Защитные заголовки | Реализовано | 4 заголовка безопасности |
| ✅ Два режима работы | Реализовано | Educational / Production |
| ✅ Разные лимиты для маршрутов | Реализовано | GET (высокий) / POST (низкий) |
| ✅ Валидация URL источников | Реализовано | Uri.IsWellFormedUriString |

### Демонстрационные сценарии

1. **Нормальная работа** — успешный запуск с корректной конфигурацией
2. **Приоритет настроек** — демонстрация переопределения (CLI > Env > File)
3. **Неверная конфигурация** — остановка запуска с понятным сообщением
4. **Недоверенный источник** — блокировка запроса с кодом 403
5. **Превышение лимита** — ответ 429 Too Many Requests
6. **Разные режимы** — подробные vs краткие сообщения об ошибках

## 📦 Установка и запуск

### Требования

- .NET 9.0 SDK или выше
- PowerShell (для тестовых скриптов)

### Клонирование и сборка

```
# Клонирование репозитория
git clone https://gitverse.ru/stasnorman/task3_framework
cd task3_framework

# Сборка проекта
dotnet build

# Запуск программы
dotnet run

# Запуск тестов
dotnet run -- --test

# Учебный режим 
dotnet run --Mode=Educational

# Боевой режим 
dotnet run --Mode=Production

```
